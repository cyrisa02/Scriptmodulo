Si vous voyez ce message en vert foncé dans un encart vert alors bravo, vous avez réussi à installer Bootstrap ! Bien se mettre dans le repertoire! Procédure pour installer Bootstrap sans CDN 1/ Télécharger sur le site la version « Compiled CSS and JS » https://getbootstrap.com/docs/4.4/getting-started/download/ Décompresser ce fichier zip dans notre répertoire de travail 2/ dans le repertoire de travail, faire un npm -> npm install jquery 3/ Bien mettre tous les scripts !
